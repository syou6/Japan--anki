# スクリーンショット配置ガイド

このフォルダに以下の画像ファイルを配置してください：

## 必要な画像ファイル
- screen1.png - メイン画面（左側中央のスマホ）
- screen2.png - 上部中央のスマホ
- screen3.png - 左下のスマホ
- screen4.png - 右側中央のスマホ
- screen5.png - 中央右のスマホ
- screen6.png - 上部右のスマホ
- screen7.png - 右端上部のスマホ

## 推奨画像サイズ
- 幅: 280px
- 高さ: 606px
（適切なサイズに調整済み）

## 画像形式
PNG または JPG形式を推奨

## 配置後の確認
ブラウザで http://localhost:5173/lp2.html を開いて確認してください。
